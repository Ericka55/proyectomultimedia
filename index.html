Filtro · Suavizado

Filtro de Promedio 3×3 Píxeles

Implementación de un filtro de promedio (mean filter) que recorre la imagen completa usando ventanas deslizantes de 3×3 píxeles. Cada píxel de salida resulta del promedio de sus 9 vecinos, reduciendo ruido y suavizando transiciones abruptas de color sin librerías de convolución externas.

4. Informe Técnico

Metodología y
herramientas empleadas

Documentación del proceso de desarrollo, tecnologías utilizadas y metodología aplicada en cada actividad.

Metodología

  • Acceso directo a arreglos de píxeles (R, G, B) sin abstracción
  • Clasificación de texturas por análisis estadístico de vecindad local
  • Convolución manual con kernel 3×3 para el filtro de suavizado
  • Manejo de bordes con padding (replicación de píxeles extremos)
  • Sincronización audio-animación por mapeo de beats a fotogramas
  • Pruebas visuales comparando salida esperada vs. obtenida

Herramientas y Tecnologías

  • Python 3 · procesamiento de imagen a nivel píxel
  • PIL / Pillow · lectura y escritura de archivos de imagen
  • NumPy · manipulación eficiente de arreglos de píxeles
  • Software de animación · producción multimedia (ejercicio c)
  • Editor de audio · sincronización ritmo-animación
  • GitHub · control de versiones y entrega del código
  • HTML/CSS/JS · plataforma web de presentación

Resultados Obtenidos

  • Clasificador capaz de identificar ≥4 tipos de textura superficial
  • Filtro 3×3 reduce el ruido salt & pepper en ≈80% visual
  • Producción multimedia completada con sincronización beat-frame
  • Código documentado con instrucciones de instalación y ejecución
  • Comparación visual antes/después generada automáticamente

Fórmula del Filtro 3×3

  • P(x,y) = Σ P(i,j) / 9, para i,j en ventana 3×3
  • Cada canal RGB procesado de forma independiente
  • Recorrido por filas y columnas con índice desplazado
  • Borde tratado con clamp al píxel más cercano disponible
  • Resultado escrito en imagen de salida nueva (no in-place)
  • Complejidad O(N·M) donde N×M = resolución de la imagen
